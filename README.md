# SAKSHAM.
> Ideas don't stay imaginary for long around here.

![Portfolio Preview](./public/favicon.svg) <!-- Keep this as a placeholder, or update if a screenshot is added -->

A personal portfolio built from the ground up to be bold, fast, and unapologetic. Designed with an aggressive geometric aesthetic, a system of curiosity, and heavy scroll-driven animations.

## Tech Stack
- **Framework:** React 18
- **Language:** TypeScript
- **Build Tool:** Vite
- **Styling:** Vanilla CSS + Tailwind CSS (for utility layouts)
- **Animation:** GSAP (ScrollTrigger, timelines) + Framer Motion
- **Typography:** Space Grotesk & Amatic SC

## The Vibe
This site rejects the traditional "clean SaaS" look. It embraces chaos, geometry, and raw, handwritten energy. Features include:
- **GSAP ScrollTrigger:** Every section reveals itself organically as you scroll.
- **Custom SVG Geometry:** The `GeometricLayer` system builds shards, hatching strokes, and intersection dots using pure math and SVG.
- **No Dependencies Where It Counts:** Minimal bloated libraries. All custom cursors, tooltips, and case study expansions were built by hand.
- **Responsive by Design:** Aggressive desktop layouts scale down gracefully to single-column mobile views without losing their edge.

## Running Locally

1. Clone the repository:
   ```bash
   git clone https://github.com/SakshamBajpayi/portfolio.git
   cd portfolio
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Build for production:
   ```bash
   npm run build
   ```
